# Comparison

## Usages

- Equal to: `==`

```python 
a==b
```

- Not Equal to: `!=`

```python 
a != b
```

- Greater Than: `<`

```python 
a < b
```

- Less Than: `>`

```python 
a > b
```

- Greater Than or Equal to: `<=`

```python 
a <= b
```

- Less Than or Equal to: `>=`

```python 
a >= b
```
